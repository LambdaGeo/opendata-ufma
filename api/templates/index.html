Dados Abertos UFMA (não oficial)

Este é um portal não oficial de dados abertos com o objetivo disponibilizar endpoints rest para acessar os dados públicos já existentes nos portais da Universidade Federal do Maranhão. Dados abertos é um paradigma que busca tornar os dados mais acessíveis a algoritmos, tornando fácil o desenvolvimento de aplicativos e análise de dados.

A Universidade Federal do Maranhão já tem disponibilizado alguns dos seus dados neste paradigma, através do portal de dados abertos: http://dadosabertos.ufma.br/ , foi exposto vinte conjuntos de dados divididos em sete diferentes grupos, como ensino, administração, extensão, pesquisa entre outros. Esse portal busca agregar com mais dados, aos já disponíveis no portal oficial. Atualmente são duas APIs, e estão documentadas abaixo:

API

Essa API retorna dados que já foram importados previamente para uma base de dados. Assim ela irá retornar os dados com mais eficiência. Só as operações POST requer autorização.

LiveAPI

A liveapi faz as consultas acessando diretamente os dados que estão no portal da universidade. Ela tem a vantagem de pegar dados sempre atualizados, e a desvantagem de ter um custo maior de tempo.
